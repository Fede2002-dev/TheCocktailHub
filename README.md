# TheCocktailHub
This is a cocktail's app builded for android with kotlin. This app is built with:  
- Coroutines
- MVVM
- Extension Functions
- Retrofit
- Room
- Navigation Components
- ViewBinding
## Architecture
Clean architecture with MVVM pattern.  
**Layers:**  
- Framework  
- Data  
- Domain  
- Use cases
